### 单一职责

* 应该有且仅有一个原因引起类的变更，简单点说，一个类，最好只负责一件事，只有一个引起它变化的原因

### 如何监听model变化

* 观察者模式

  * addEventListener就是一个观察者模式

    ```js
    target.addEventListener("click",listener);
    target.dispatchEvent("click");
    ```

  * 现在自己引入一个类似的事件监听器

    ```js
    let event = new EventEmitter();
    event.on('addItem',showlist);
    event.emit('addItem');
    ```
