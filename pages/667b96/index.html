# 数组迭代器方法

  1. array.keys(): 返回数组中的键名（迭代器）
  2. array.values(): 返回数组中的值（迭代器）
  3. array.entries() : 返回数组中的键值对（迭代器）
        const a = ["q","w","e"];
        console.log(a.keys());
        console.log(a.entries());
        console.log(a.values());

        //用 for of 对迭代器对象进行遍历
        for(const [key,value] of a.entries()){
            console.log(key,value);
        }

# 数组的复制和填充

  1. array.fill( value , begin , end): 向一个已有数组插入全部或部分相同的值(不包括 end )
        const b = [0,0,0,0];

        console.log(b.fill(6));//[6,6,6,6]

        console.log(b.fill(7,1));//[6,7,7,7]

        console.log(b.fill(8,2,3));//[6,7,8,7]

  1. array.copyWithin(): 按照指定范围浅复制数组中的部分内容（不包括 end ）

# 转换方法

  1. array.toString()：返回由数组中每个值等效的字符串拼接而成的一个由逗号分隔的字符串，每个值都调用toString() 方法
  2. array.toLocaleString()：和 toString() 不同的是，每个值都调用 toLocaleString() 方法
  3. array.valueOf()：返回数组本身
  4. array.join( 字符串分隔符 )：返回一个由 字符串分隔符 连接而成的字符串

# 栈方法

  1. array.push(...args): 接受任意数量的参数，并把他们添加到数组末尾
  2. array.pop(): 返回数组中最后一项元素，并删除该元素，数组长度减一

# 队列方法

  1. array.push(...args): 接受任意数量的参数，并添加到数组末尾
  2. array.shift(): 取得数组中第一个元素，并删除该元素，数组长度减一
  3. array.unshift(): 取得数组中最后一个元素，并删除该元素，数组长度减一

# 排序方法

  1. array.reverse(): 将元素反向排列,反转数组
  2. array.sort( function ): 排序方法，参数是一个函数
        const c = [{name:"english",value:"60"},{name:"chinese",value:"50"},{name:"math",value:"55"}];

        //升序
        c.sort((a,b)=>{
            if(a.value > b.value){
                return 1;
            }else{
                return -1;
            }
        });

# 操作方法

  1. array.concat(...args): 创建当前数组的一个副本，把它的参数添加在副部末尾，返回新构建的数组
  2. array.slice(begin, end ):创建一个包含原有数组中一个或多个元素的新数组，返回从 begin 索引到 end(可选) 索引对应的所有元素，但不包括 end 元素，不影响原始数组
  3. array.splice()：
  • 删除: splice(index,num) 删除第 index 位置开始 num 个元素
   const d = [1,2,3];
   d.splice(1,1); //[1,3]
1
2
  • 插入: splice( index , 0 , ...args ) 找到指定 index 插入多个元素
  • 替换: splice( index , num , ...args) 删除从 index 位置开始的 num 个元素，然后从该 index 位置插入 多个元素

# 搜索和位置方法

# 严格搜索

  1. array.indexOf(): 从头开始找位置
  2. array.lastIndexOf(): 从后往前找位置
  3. array.includes(): 返回boolean值，表示是否至少找到一个与指定元素匹配的项

# 断言函数

  1. array.find( function ): 返回第一个匹配的元素
  2. array.findIndex( function ): 返回第一个匹配元素的索引

# 迭代函数

  1. array.every( callback(element,index,array) ): 对数组中每一项都执行 callback 函数，如果都返回true ，则方法最后返回 true
       const iter = [1,2,3,4,5,6];
       const iterFalse = iter.every((element,index,array)=>{
           return element>=2;
       });

       const iterTrue = iter.every((element,index,array)=>{
           return element > 0;
       });
       console.log(iterFalse);
       console.log(iterTrue);
